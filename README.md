# Wayfarer Beta

Wayfarer is a local-first multimodal AI assistant for research, writing, source-grounded conversations, and document analysis.

## Download

Download the latest Windows beta from the [Releases page](https://github.com/Nachlassian/bridge-wayfarer-beta/releases).

## Requirements

- Windows 10 or Windows 11, 64-bit
- A compatible local model through LM Studio for AI chat

## Beta notice

Wayfarer is currently beta software. Features and data formats may change during testing.

Wayfarer does not automatically upload support information or project content.
